# localshow
 My echo show replacement dashboard

## Prerequisites

This assumes there's a homeassistant instance running and accessible at http://192.168.6.168:8123. See all Home Assistant documentation for configuring and running a homeassistant instance.

The Anylist integration is provided by https://github.com/kevdliu/hassio-addon-anylist and https://github.com/kevdliu/hacs-anylist.
